Contents

This directory contains a bunch of files tested with Coq-8.5beta2. It is structured as follows:

16 directories, each one associated with a chapter of the book:

Each directory contains a list of exercises, and an adaptation to CoqV8.5 of the scripts presented in the corresponding chapter of the book. Each index file contains also corrections of the errors that were kindly signalled to us by the readers.

Sources and exercises from the book

This site contains the source of all examples and the solution of 170 over 200 exercises from the book. Some new exercises are still added to this repository.
For each exercise, we give a solution as a Coq file, together with some comments if the exercise is difficult, or if the solution presents some methodological interest. Comments are welcome.
